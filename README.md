## ✨ 核心亮点

本项目专为**教室投影仪大屏幕**与**一线教师**设计，彻底改变传统的单向 PPT 教学：

- 🧊 **冰雪直连魔法**：内置强制系统代理清理机制，**纯国内网络直连**，极速秒开！
- 📐 **完美数学渲染**：全面支持 `LaTeX` 语法，无论是微积分还是化学方程式，都能在大屏上完美渲染。
- 🗺️ **文科导图生成**：内置 `Mermaid.js` 引擎，一键将枯燥的政史地 PPT 提炼为**可视化的树状思维导图**。

## 🛠️ 如何在本地运行？

### 1. 安装依赖
确保你的电脑已安装 Python 3.8+。打开终端运行：
```bash
pip install -r requirements.txt
```

### 2. 准备背景
请确保 `cirno.jpg` 与代码在同一目录下，否则天才⑨是不会出现的哦！

### 3. 运行程序
```bash
python ppt_ai_assistant.py
```
*(运行后，在软件顶部输入你的 ChatAnywhere API Key 即可启动魔法！)*
